To run this application on your local machine, you need Python and the NiceGUI package installed.

### Prerequisites

* Python 3.8+

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/rahuljangra396/NiceGUI-Task_Manager.git](https://github.com/rahuljangra396/NiceGUI-Task_Manager.git)
    cd NiceGUI-Task_Manager
    ```

2.  **Create and activate a virtual environment (Recommended):**
    ```bash
    python -m venv venv
    # On Windows (in PowerShell/CMD):
    .\venv\Scripts\activate
    # On Linux/macOS (in Bash/Zsh):
    source venv/bin/activate
    ```

3.  **Install NiceGUI dependency:**
    ```bash
    pip install nicegui
    ```
